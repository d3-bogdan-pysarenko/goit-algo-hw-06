## Висновок:

Для виконання практичного завдання 2 було використано алгоритми Depth-first search (DFS) та Breadth-first search (BFS) для знаходження шляху між двома найбільш віддаленими вершинами у графі. Отримано наступні резудьтати:

### Пошук шляху за допомогою Depth-first search (DFS)
```bash
DFS шлях: ['Hospital', 'West_Market', 'Center', 'North_Square', 'Hill', 'East_Park', 'School']
```

### Пошук щляху за допомогою Breadth-first search (BFS)
```bash
BFS шлях: ['Hospital', 'West_Market', 'Center', 'East_Park', 'School']
```

Враховуючи результати, можемо зазначити що алгоритм Breadth-first search (BFS) є більш ефективним з точки зору меншої кількості кроків для знаходження маршруту між двома найбільш віддаленими вершинами циклічного графу у порівнянні з Depth-first search (DFS).
Вказане можна пояснити самою "природою" вказаних алгоритмів, через що BFS майже завжди буде ефективнішим у знаходженні найкоротшого шляху у циклічних незважених графах подібного типу. Однак DFS може бути ефективнішим у випадку пошуку шляху у деревоподібному незацикленому графі з високою ступінню заглиьленності в порівнянні з BFS.
Загалом ефективність дуже залежить від конкретного випалку і задач, які мають бути вірішені.
